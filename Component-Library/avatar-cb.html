AVATARS
  • Circular Avatars
    The avatar component is typically used to display circular user profile pictures. It is typically available in 5 sizes: avatar-xl(5rem), avatar-large(4rem), avatar-md(3rem), avatar-small(2rem), avatar-xs(1.5rem).
  • Sqaure Avatars
    The avatar component is typically used to display square user profile pictures. It is typically available in 5 sizes: avatar-xl(5rem), avatar-large(4rem), avatar-md(3rem), avatar-small(2rem), avatar-xs(1.5rem).
  • Avatars with Status
    The avatar component is typically used to display square/circle user profile pictures along with status indicators. There are typically 3 status indicators: online, offline and busy.
