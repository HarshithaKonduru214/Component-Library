BUTTONS

<link rel="stylesheet" href="button.css">
  • Solid Buttons
    This component is typically used to display solid buttons that are essentially used for primary actions.
    
<div class="button-solid">
    <button class="margin button primary-button">Primary Button</button>
    <button class="margin button secondary-button">Secondary Button</button>
    <button class="margin button danger-button">Danger Button</button>
</div>
  • Outlined Buttons
    This component is used to display outlined buttons that can be used as secondary buttons.
    
<div class="button-outlined">
    <button class="margin button primary-button-outlined">Primary Button</button>
    <button class="margin button secondary-button-outlined">Secondary Button</button>
    <button class="margin button danger-button-outlined">Danger Button</button>
</div>
  • Button with Icons
    This component is used to display buttons that have icons.
    
<div class="button-icon flex-row">
    <button class="margin button primary-button flex-row flex-align">
        <img class="icon" src="cart-white.svg" alt="icon">
        <span>Add to Cart</span>
    </button>
    <button class="margin button danger-button flex-row flex-align">
        <img class="icon" src="delete_white.svg" alt="icon">
        <span>Delete</span>
    </button>
</div>
  • Floating Buttons
    This component is used to display floating buttons.
    
<div class="button-float">
    <button class="margin float-button primary-button">
        <img src="add_white.svg" alt="Icon">
    </button>
</div>
