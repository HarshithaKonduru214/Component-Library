INTRODUCTION
Oxygen UI is a customisable light weight CSS library which provides basic components required to build any website.

  • Installation
    To start using the components of Oxygen UI you can import the below stylesheet in your project. 
    
<link rel="stylesheet" href="/style.css">
    NOTE: If you do not require all the components, you can also import individual component stylesheets. The links to the individual component stylesheets can be found in their respective documentations.
