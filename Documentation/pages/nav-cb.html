Navigation Bar
A navigation bar is a link to appropriate sections/pages in a website that helps readers in traversing the online document. Import the below stylesheet in your project to use the navbars given below. 

@import url("https://oxygen-ui-component-library.netlify.app/components/nav/nav.css");
  • Simple Nav Bar
    This is a simple navbar with just the name and social media links. Copy the below code to use this component.
    OXYGEN UI
    
<nav class="navbar mb-2 pt-2 flex-row box-shadow justify-content-between">
