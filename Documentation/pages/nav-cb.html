Navigation Bar
A navigation bar is a link to appropriate sections/pages in a website that helps readers in traversing the online document. Import the below stylesheet in your project to use the navbars given below. 

@import url("https://oxygen-ui-component-library.netlify.app/components/nav/nav.css");
  • Simple Nav Bar
    This is a simple navbar with just the name and social media links. Copy the below code to use this component.
    OXYGEN UI
    
<nav class="navbar margin-bottom padding-top flex-row box-shadow">
    <div class="nav-title align-center-item">OXYGEN UI</div>
    <div class="nav-buttons flex-row">
        <a href="github-link" target="__blank" class="nav-icon button margin"><img src="/Documentation/images/github-brands.svg" /></a>
        <a href="twitter-link" target="__blank" class="nav-icon button margin"><img src="/Documentation/images/twitter-brands.svg" /></a>
        <a href="linkedin-link" target="__blank" class="nav-icon button margin"><img src="/Documentation/images/linkedin-brands.svg" /></a>
    </div>
</nav>
